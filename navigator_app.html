🧠 Test Your Knowledge
You tested the same prompt across different Gemini models in Google AI Studio and got noticeably different outputs — if you were a chef tasting two wines from the same vineyard but different vintages, what specifically would you be looking for to decide which model to use for a customer-facing AI product, and why does the system instruction act as the sommelier in this analogy?
You should be evaluating tone consistency, hallucination risk, response length control, and how literally each model follows nuanced instructions. The system instruction is your sommelier because it frames the context before any user input arrives — without it you're letting the model freelance, which is fine for experimentation but reckless for production.
